== Description ==

VC MailChimp plugin helps you build the MailChimp subscribe forms very easy with WPBakery Page Builder (formerly Visual Composer). You can add many MailChimp accounts on one site, choose the list and style for this form as you want.

= Live demo =

Click to see [live demo](http://demo.wpclever.net/?item=vcmc "live demo")

= Features =

- Unlimited MailChimp accounts in one site
- Fetch and select the list at glance
- Support custom fields
- Easy custom layout via WPBakery Page Builder (formerly Visual Composer)
- Easy change style to fit with your theme
- WPML integration

= Translators =

Available Languages

- English (Default)

If you have created your own language pack, or have an update for an existing one, you can send [gettext PO and MO file](http://codex.wordpress.org/Translating_WordPress "Translating WordPress") to [us](https://wpclever.net/contact "WPclever.net") so we can bundle it into VC MailChimp.

== Installation ==

1. Please make sure that you installed WPBakery Page Builder (formerly Visual Composer)
2. Unzip the vc-mailchimp.zip
3. Copy vc-mailchimp folder to wp-content/plugins
4. Go to Plugins / Installed Plugins, find VC MailChimp and click Activate
5. Now you can add the MailChimp form when add/edit the page with WPBakery Page Builder
